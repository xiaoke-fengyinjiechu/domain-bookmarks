# 域名收藏夹

一个智能的 Chrome 浏览器收藏夹管理扩展，按域名分组管理您的书签。

## 功能特性

### 核心功能
- **域名分组**：自动按一级域名将收藏夹分组，便于管理
- **当前域名收藏**：访问页面时自动显示相同域名的收藏夹（支持端口匹配）
- **收藏管理**：支持添加、编辑、删除收藏夹
- **智能搜索**：快速搜索收藏夹，支持标题和关键词搜索
- **网站图标**：自动显示每个收藏夹的网站图标

### 界面特性
- **左右布局**：左侧显示域名分组，右侧显示收藏夹列表
- **交互友好**：点击域名分组可切换显示该组收藏，再次点击取消选中显示所有收藏
- **响应式设计**：简洁美观的界面，易于使用

## 安装方法

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`

2. 开启右上角的"开发者模式"

3. 点击"加载已解压的扩展程序"

4. 选择此项目的根目录

5. 扩展安装完成！

## 使用说明

### Popup 页面
1. 点击浏览器工具栏中的扩展图标打开 Popup 页面
2. **所有收藏**标签页：
   - 左侧显示按一级域名分组的收藏夹
   - 点击域名分组可查看该组下的收藏夹
   - 再次点击取消选中，显示所有收藏
   - 支持搜索收藏夹
3. **当前域名**标签页：
   - 显示当前访问页面的收藏夹（含端口匹配）
   - 支持编辑和删除操作
4. **添加收藏**：点击"添加收藏"按钮添加当前页面到收藏夹

### 页面注入
1. 访问任意网页
2. 点击页面右上角的"🔖 收藏夹"按钮
3. 自动显示当前域名的收藏夹列表（含端口匹配）
4. 点击收藏夹链接快速跳转

### 收藏管理
- **添加收藏**：在 Popup 页面点击"添加收藏"按钮
- **编辑收藏**：点击收藏项的"编辑"按钮修改标题和 URL
- **删除收藏**：点击收藏项的"删除"按钮删除收藏

## 技术栈

- **Manifest V3**：Chrome 扩展最新标准
- **JavaScript (ES6+)**：现代 JavaScript 语法
- **CSS3**：响应式布局和动画效果
- **Chrome Extension APIs**：
  - Bookmarks API：收藏夹管理
  - Tabs API：标签页信息获取
  - Storage API：数据存储
  - Runtime API：消息传递

## 开发说明

- 修改代码后，在 `chrome://extensions/` 页面点击"刷新"按钮重新加载扩展
- 使用 Chrome DevTools 调试 Popup 和 Content Script
- 查看 Console 日志了解扩展运行状态
- Service Worker 日志在 `chrome://extensions/` 页面的"Service Worker"链接中查看

## 注意事项

- 收藏夹分组按一级域名进行（如 baidu.com、github.com）
- 当前域名匹配包含端口信息（如 localhost:3000）
- 网站图标从各网站根目录的 favicon.ico 获取
- 扩展需要"书签"和"标签页"权限才能正常工作
